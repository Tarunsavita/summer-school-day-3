Cubic-Bezier-Curve

A cubic Bezier curve is a parametric curve defined by four points: a start point, an end point, and two control points. It runs from the start point towards the first control point and bends to end at the end point, with the control points determining the direction of the curve at its ends. Cubic Bezier curves are widely used in computer graphics and animations to create smooth transitions and shapes. A cubic Bézier curve is a type of parametric curve widely used in computer graphics, UI animations, and CSS transitions to create smooth, natural-looking motion. It’s defined by four points: P₀: the starting point of the curve P₁ and P₂: control points that shape the curve’s direction and bend P₃: the ending point of the curve The curve doesn’t pass through the control points (P₁ and P₂), but they influence its shape—like magnets pulling the curve toward them.
